PetResgate 🐾
O PetResgate é um sistema computacional monolítico com arquitetura baseada em API REST voltado para a modernização e digitalização dos processos operacionais de Organizações Não Governamentais (ONGs) e protetores independentes de animais.

🏢 Sobre a ONG e o Problema
Atualmente, a gestão de animais resgatados, prontuários clínicos, triagem de processos de adoção e o controle de insumos/doações recebidas ocorrem de forma descentralizada (por planilhas ou papel). O PetResgate centraliza essas operações em uma plataforma web responsiva, mitigando a perda de históricos médicos, evitando duplicidade de cadastros de adotantes e trazendo previsibilidade para o estoque e fluxo de doações.

🛠️ Stack Tecnológica
Frontend: React.js (v18.x)
Backend: Spring Boot (Java v3.2.x) com Hibernate (JPA)
Banco de Dados: PostgreSQL (v15.x)
Segurança: Spring Security + JWT (JSON Web Token) + BCrypt
Armazenamento de Arquivos: Amazon S3
